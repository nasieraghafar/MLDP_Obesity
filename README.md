# Obesity Predictor using Machine Learning

Used Jupyter Notebook

Steps Taken: train test split (70-30), check for missing values, list of unique values, check for inconsistencies in dataset, identify outliers, one-hot encoding, feature scaling, label encoding, grid search, feature importances, confusion matrix

Use of graphs: Histograms, Boxplot, Correlation HeatMap, Scatter Plot

Models  tested: Logistic Regression, Random Forest Classifier, K-Nearest Neighbours (KNN), Multi-Layer Perceptron (MLP), Support Vector Machine (SVM).
